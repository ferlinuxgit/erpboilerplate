# Roadmap ERP

Este documento resume la implementación por fases del ERP:

- Fase 0: plataforma (`tenant/company/fiscalYear`, RBAC, auditoría).
- Fase 1: maestros.
- Fase 2: ventas.
- Fase 3: compras.
- Fase 4: inventario.
- Fase 5: contabilidad.
- Fase 6: tesorería.
- Fase 7: fiscal.
- Fase 8: reporting.
- Fase 9: operación SaaS.
- Fase 10: hardening.
- Fase 11: internacionalización.
- Fase 12: onboarding/equipo.
- Fase 13: Stripe/billing.
- Fase 14: emails y adjuntos.
- Fase 15: PDFs y exportes.
- Fase 16: calidad, tests y CI/CD.
